# /antigravity — Bootstrap de Runtime Cognitivo

**Description**: Inicializa una estructura de directorio .antigravity/ de grado de producción, transformando el IDE en un runtime cognitivo con memoria, control de costes y agentes especializados.

## Steps

1. **Initialize Root**: Crea la carpeta `.antigravity/` en el directorio actual.
2. **Deploy 9-Layer Architecture**: Genera las subcarpetas de Config, Runtime, Memory, Workspace, Agents, Skills, Tools, Logs y Policies.
3. **Setup Logic Placeholders**: Crea los archivos base (`router.py`, `cost_controller.py`, etc.) con la lógica de gobernanza cognitiva.
4. **Configure Policies**: Establece los límites de uso de LLM y ejecución determinista.
5. **Finalize with Bootstrap**: Crea el script `bootstrap.py` para la orquestación del sistema.

## Principles
- Prioriza la ejecución local y determinista sobre el uso de LLM.
- Implementa una memoria operativa persistente.
- Controla el presupuesto de tokens de forma activa.
